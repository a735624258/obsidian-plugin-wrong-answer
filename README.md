# Obsidian 错题卡片

多选题批改与闪卡复习。用代码块语法在笔记中插入可翻转的错题卡片。

## 语法

````markdown
```wrong-card
front: 题目截图.png
back: 答案截图.png
note: 这道题选B，因为……
```
````

## 功能

- 正面显示题目截图，背面显示答案截图
- 点击卡片翻转查看答案
- 右侧显示解析/笔记
- 适合考试刷题场景：截图错题 → 贴代码块 → 翻转复习

## 使用

1. 截图错题和答案
2. 在笔记中插入 `wrong-card` 代码块
3. 阅读模式下点击卡片翻转

## 安装

下载 `main.js`、`manifest.json`、`styles.css`，放入 `.obsidian/plugins/wrong-answer/`，重启 Obsidian 启用。
